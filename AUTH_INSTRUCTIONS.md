# Инструкция по авторизации в Telegram

Для работы скрипта требуется одноразовая авторизация в Telegram.

## Шаг 1: Авторизация

Запустите скрипт авторизации в терминале (интерактивно):

```bash
source venv/bin/activate
python auth_telegram.py
```

Скрипт запросит:
1. **Номер телефона** (в международном формате, например: +79991234567)
2. **Код подтверждения** (придет в Telegram)

После успешной авторизации сессия сохранится в файл `session.session`.

## Шаг 2: Запуск основного скрипта

После авторизации основной скрипт можно запускать автоматически:

```bash
source venv/bin/activate
python main.py
```

Сессия будет использоваться автоматически, повторная авторизация не потребуется.

---

**Примечание:** Если файл `session.session` существует, но авторизация не работает, удалите его и запустите авторизацию заново:

```bash
rm session.session
python auth_telegram.py
```
